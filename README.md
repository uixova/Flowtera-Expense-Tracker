# 🌊 Flowtera | Smart Expense & Trip Tracker

Flowtera is a modern, Single Page Application (SPA) designed for businesses and individuals to track expenses, 
manage business trips, and analyze financial flows with a sleek, high-performance interface.

## ✨ Features

- SPA Architecture: Seamless page transitions without refreshing, powered by a custom JavaScript router.
- Expense Management: Detailed tracking with categories, merchants, and status badges.
- Trip Tracking: Specialized module for travel management with vehicle-specific icons.
- Modern UI/UX: Dark-themed, responsive design built with CSS Grid and Flexbox.
- Role-Based Concept: Designed with Admin/User permission logic.

## 🛠️ Tech Stack

- Frontend: HTML5, CSS3 (Grid & Flexbox), Vanilla JavaScript
- Icons: Tabler Icons
- Charts: Chart.js (Planned)
- Backend: Node.js & Express (In development)

## 🚀 Getting Started

1. Clone the repository:
   git clone https://github.com/uixova/Flowtera-Expense-Tracker.git

2. Navigate to project:
   cd Flowtera-Expense-Tracker

3. Install dependencies:
   npm install

4. Run the server:
   npm start
